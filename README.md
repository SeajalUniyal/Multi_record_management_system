# Multi_record_management_system

A beginner-friendly, web-based data reporting system built with **PHP**, **HTML**, and **MySQL**. This project allows users to input and manage records for multiple entities including **students**, **employees**, **passengers**, and **shopkeeper bills** using dynamic forms and simple database operations.
